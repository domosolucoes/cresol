# Cresol - Migração de perfil de usuários

## Descrição

Script em Shell Bash para migrações de perfil de usuário Debian e Ubuntu

## Getting Started

### Dependências

* wget
* Shell Bash

### Instalação

* wget script.sh
* chmod +x script.sh

### Executando o programa

```
./script.sh
```
